DESKTOP ASSISTANT

I am currently working on a Desktop Assistant - "Rachel" as a personal project.

It will be able to do several tasks with ease and flexbility. It currently has been implemented with actions that include opening applications, meanings of words, customized user commands, direct bash commands and multiple commands in one statement. Integrated Natural Language Processing to make clear distinctions between sentences like - "open firefox" and "what is the meaning of open?".

I am currently working on fetching news, stock data, weather and mails. Also will be merging it with SpeechRecognition API to invoke Rachel on one call.
